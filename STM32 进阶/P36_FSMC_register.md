# 电路原理

FSMC存储器扩展  STM32F103ZET6开发板 预留了大量的存储器插槽  提供给用户自己进行扩展

BANK1_NE3  =>  SRAM    =>  0X68000000



A0-A18  => 地址线

D0-D15 => 数据线

// 在使用的时候  使能片选线属于长期有效

NOE => 输出使能

NCS1 => NE3

NWE => 写使能



# 案例实现

## 扩展地址使用

(1) 使用关键字定义

```c
// 32位对齐  地址需要是4的整倍数
uint8_t v1 __attribute__((at(0x68000000)));

// 关键字定义的方式在函数方法中不生效  =>   会被自动优化到系统的RAM中
```

(2) 直接使用指针

```c
// 直接对指针进行赋值  => 不推荐  代码没有可读性
*(uint8_t *)0x68000001 = 15;

// 使用宏定义  给指针起名字  =>  可读性比较强
#define V4 ((uint8_t *)0x68000002)

*V4 = 24;
printf("v4:%p,%d\n",V4,*V4);
```
