# 电路原理

使用ADC1的通道10连接引脚PC0  =>  连接可变电阻  =>   PC0输入了一个可变的模拟信号  可以测量引脚对应的电压
设置参考电压为3.3v

测量结果为12为数字  x   

测量最终的电压就是  x * 3.3v / 4095

# 内部寄存器

(1) SR: 中断标志位记录   开始转换  转换完成  看门狗

(2) CR1: 选择ADC通道模式  选择看门狗模式   **选择扫描模式**

(3) CR2: 内部的温度传感器和标准电压测量  **软件触发功能选择   左右对齐  DMA功能  校准  连续转换 上电**  

(4) **SMPR:采样时间通道寄存器  =>  记录每一个通道的采样时间**

(5) JOFR: 注入通道偏移量寄存器

(6) 看门狗电压上限寄存器    看门狗电压下限寄存器   

(7) **SQR123: 规则序列寄存器  =>  将需要转换的通道存放到序列中  同时添加一个转换序列的个数长度**

(8) JSQR 注入通道规则序列    JDR 注入通道数据寄存器

(9) DR数据寄存器  =>  **低16位按照数据对齐选择存到最终转换数据**

# 案例实现

## ADC1的初始化

(1) 初始化时钟ADC1   =>  ADC使用的时钟小于14MHz  需要使用预分频6分配   

(2) 配置PC0的引脚模式  =>  模拟输入0000

(3) 数据对齐    扫描模式    连续转换

(4) 采样时间

(5) 配置通道序列  添加10号通道到序列中   

(6) 填写转换通道的个数为1

## 开始转换的流程

(1) ADON 上电

(2) AD校验 => 等待校验完成

(3) 开启转换

 	1. 二次上电完成开启转换
 	2. 外部触发 软件模式SWSTART 

(4) 等待第一次转换完成  

## 读取转换之后的电压值

ADC1->DR  存储的值是12位的数字 

DR * 3.3 / 4095
