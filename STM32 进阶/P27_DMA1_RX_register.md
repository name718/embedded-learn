# 电路原理

DMA是系统自带的传输功能  =>  不经过CPU  使用DMA可以减轻CPU的压力  同时传输的效率更高效

将USART1_RX接受到的数据  存储到内存中   =>  最好能连续接收存储

# 内部寄存器

(1) 中断标记寄存器ISR:  传输错误中断   半传输中断  传输完成中断

(2) 清除中断标记寄存器IFCR: 置位1清除对应的中断标记   CGIFx可以一次清除当前通道所有的中断标记

(3**) CCR配置寄存器: 传输的方向   优先级   地址自增  数据宽度  使能  开中断**  初始化核心配置

(4) 数据长度寄存器CNDTR : 一次传输的数据长度

(5) CPAR外设地址寄存器: 存放当前传输的外设地址   =>  USART1_DR

(6) CMAR 存储器地址寄存器: 存放当前传输的存储器地址 => array
